# RBWS — Hollow Meridian

A band website for **Hollow Meridian**, a post-rock group from Tallinn, Estonia. Built with vanilla JavaScript and [Vite](https://vite.dev/).

## About

The site promotes the band's debut album *Static Cartography* and includes:

- **Hero** — album artwork and call-to-action buttons
- **Introduction** — band bio, member profiles, and key stats
- **Gallery** — photo mosaic
- **Events** — upcoming tour dates with ticket availability
- **References** — press quotes and review scores

## Getting started

```bash
cd "Hollow Meridian"
npm install
npm run dev
```

| Command | Description |
|---------|-------------|
| `npm run dev` | Start local dev server |
| `npm run build` | Build for production |
| `npm run preview` | Preview production build |
